Help

                
How to use

                
Disassemble .bin file with AmazfitBipTools (AmazfitCorTools                    for Cor), click on Select images and select them from watchface folder by Ctrl+A, then select JSON file by clicking on JSON Select. After that you can set numbers to display by clicking Settings or edit
                    the watchface
                    

You can edit elements location on tab Design
                

                
Also you can edit JSON on tab Edit. To update preview just click out of JSON input. If you add smth by choosing it, it will use default image. You can change it to yours or download default images here
